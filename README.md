# SchoolOS: The Cognitive Operating System

**Current Version:** v0.1 (Alpha)
**Objective:** Co-evolution of Human and Artificial Intelligence.

## 1. Core Philosophy
We are not building an ERP. We are building a Biological Metasystem.
SchoolOS optimizes for **Yield**:
- **Yield of Capital:** Lucro Real (Net Revenue).
- **Yield of Space:** Asset Utilization.
- **Yield of Soul:** Cognitive Development (Resonance).

## 2. The Stack
- **Framework:** Next.js 14 (App Router)
- **UI Engine:** Mantine v7
- **Database:** PostgreSQL
- **ORM:** Prisma (Library Engine - No Cloud Proxy)

## 3. Modules
- **The Heart (Ledger):** Sequential, immutable financial tracking.
- **The Brain (Symbiosis):** Management of AI Shards assigned to students.
- **The Nervous System (Flow):** Real-time tracking of student engagement (Flow vs Anxiety).
- **The Immune System (Network):** Identification of At-Risk nodes and High-Leverage connectors.

## 4. Operational Mandates
1. **Data Sovereignty:** No data leaves the local instance without sanitization.
2. **Atomic Transactions:** No movement without a log.
3. **Human-First Interface:** Reduce cognitive load for teachers.
